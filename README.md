# Student Performance Machine Learning Project

## Overview
Predicting student academic performance using machine learning. The project shows the full data science workflow: data cleaning, visualization, modeling, evaluation, and deployment.

## What I Did
- Cleaned and preprocessed the dataset  
- Built and compared Linear Regression, Ridge Regression, and Random Forest models  
- Tuned hyperparameters using Pipelines and GridSearchCV  
- Deployed the model with Streamlit for interactive predictions  

## Tools & Technologies
- Python: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn  
- Deployment: Streamlit  
- Version Control: Git & GitHub
